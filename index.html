HTML

Principais características

  • Define a disposição de elementos como textos, links, imagens e botões
  • Permite criar e estruturar seções, parágrafos e links
  • É usado para desenvolver portais e documentos eletrônicos on-line
  • É usado em combinação com outras linguagens, como CSS, JavaScript e PHP
  • Os documentos HTML são ficheiros que terminam com a extensão .html ou .htm

História

  1. Criação (1991): Tim Berners-Lee criou o HTML para organizar e compartilhar informações na web com apenas 18 tags.
  2. Evolução Inicial (1995-1997): HTML 2.0 padronizou a linguagem e HTML 4.0 introduziu suporte a CSS e separação de estrutura e estilo
  3. HTML5 (2008-2014): HTML5 foi desenvolvido para adicionar recursos como áudio, vídeo e gráficos, finalizado em 2014
  4. Padrões e Desenvolvimento (1990 - Presente): A evolução foi guiada por organismos como o W3C e WHATWG, garantindo compatibilidade e melhorias contínuas.
  5. HTML Atual: O HTML continua sendo a base para o desenvolvimento web moderno, com novos recursos e atualizações frequentes.

O que é HTML?

HTML (HyperText Markup Language) é a linguagem usada para estruturar páginas web. Ele permite organizar o conteúdo de forma clara, como textos, imagens e links, e é a base para qualquer página na internet.

Por que aprender HTML?

-HTML é essencial para criar páginas web.

-Ele é a fundação para outras linguagens web, como CSS (estilo) e JavaScript (interatividade).

-Com HTML, você pode construir desde páginas simples até sites complexos.

Como funciona o HTML?

-HTML usa tags para organizar conteúdo: por exemplo, "h1" para títulos e "p" para parágrafos.

-A estrutura básica inclui o elemento "html", que envolve todo o conteúdo da página, e é dividido em "head" (informações sobre a página) e "body" (conteúdo visível).

-Atributos são usados para especificar detalhes adicionais, como href para links ou src para imagens.

Como começar com o HTML?

Para começar a usar HTML, você só precisa de um editor de texto simples. Ao escrever seu código HTML, lembre-se de sempre fechar suas tags corretamente e usar as estruturas corretas para garantir que o conteúdo seja renderizado adequadamente pelos navegadores.

Dica de HTML:

Sempre feche suas tags HTML corretamente para evitar erros na renderização da página
